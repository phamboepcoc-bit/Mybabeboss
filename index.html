🎉 Sẵn sàng import vào Google Calendar!
Tải file ICS rồi làm theo hướng dẫn để thêm toàn bộ lịch học vào Google Calendar.
1
Tải file ICS
Nhấn nút xanh phía trên để tải toàn bộ lịch học về máy tính.
2
Tạo Lịch mới (Rất quan trọng)
Vào Trang Tạo Lịch, đặt tên lịch rồi ấn Create calendar.
3
Import vào lịch vừa tạo
Chuyển sang menu Import & Export. Tải file ICS lên, và NHỚ CHỌN TÊN LỊCH VỪA TẠO ở mục Add to calendar trước khi ấn Import.
4
Xong! 🎉
Toàn bộ lịch học sẽ xuất hiện trên Google Calendar!
