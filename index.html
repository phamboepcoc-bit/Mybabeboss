🎉 Sẵn sàng import vào Google Calendar!
Tải file ICS rồi làm theo hướng dẫn để thêm toàn bộ lịch học vào Google Calendar.
🔗 Mở Google Calendar Import
1
Tải file ICS về máy
Bấm nút "Tải file ThoiKhoaBieu.ics" ở trên.
2
Mở Google Calendar Settings
Vào calendar.google.com → Biểu tượng ⚙️ → Cài đặt → menu trái chọn Import & Export
3
Chọn file và Import
Bấm "Select file from your computer" → chọn file ThoiKhoaBieu.ics → bấm Import.
4
Xong! 🎊
Toàn bộ lịch học sẽ xuất hiện trên Google Calendar ngay lập tức!
